Skok Strategy Engine
This is a repo for strategic tools (vibecoding)

An AI-augmented strategy tool based on Michael Skok's Value Proposition Framework (Harvard Innovation Lab). Built as part of a Master's course in Product Strategy — exploring the shift from manual workshop tools to Human + AI strategy orchestration.

The Three Methods
4U — Is the problem worth solving? Scores Unworkable, Unavoidable, Urgent, Underserved (1–5 each).
3D — Is the solution a real breakthrough? Scores Discontinuous, Defensible, Disruptive (1–5 each). Classifies moat type and delivers a final verdict: 3D APPROVED / CONDITIONAL / PIVOT / NOT APPROVED.
BLAC&White — What type of problem is it? Classifies Blatant vs. Latent and Aspirational vs. Critical. Evaluates white space opportunity.
Each method is a standalone autonomous agent prompt for Claude — human provides input, AI delivers the full analysis.

Framework
Based on Startup Secrets by Michael Skok, Harvard Innovation Lab. AI engine: Claude by Anthropic.
